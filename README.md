# Firewall: 

## Introduction

A firewall is a network security system that monitors and controls traffic based on some rules and acting as a barrier between trusted and untrusted networks.

## Types of Firewalls

### Packet Filtering

Basics filters using IP addresses and ports.The filtering uses IP addresses and ports. If a packet matches an allow rule it is permitted to pass through the firewall. Otherwise it is blocked. 


### Stateful Inspection

Tracks active connection contexts for better security.This allows them to distinguish between legitimate packets for established connections and other malicious packets attempting to exploit the network.

### Application Layer

Inspects packet content to block specific application-level attacks. They can detect and block malicious content, malware, and attacks like SQL injection and cross-site scripting that might pass through lower-level firewalls

## Rules and Importance


The real strength of a firewall comes from how well it is designed and managed. Instead of allowing everything and reacting later a good firewall blocks all traffic by default and only permits what is truly needed.
      This approach greatly reduces security risks. Since firewalls sit at the edge of a network they play a crucial role in stopping attacks early.
